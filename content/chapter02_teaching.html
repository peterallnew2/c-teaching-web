2-1 C語言的特性

C語言是一種高階、結構化的程式語言，由美國貝爾實驗室的丹尼斯·里奇（Dennis Ritchie）於1972年所開發。它結合了高階語言的易用性與低階語言的系統控制能力。

  • 簡潔有力： C語言的語法相對簡單，關鍵字數量不多，適合初學者入門。
  • 執行效率高： C語言編譯後的程式執行速度接近組合語言，適用於系統程式設計。
  • 功能豐富： 提供了多樣的運算子和資料型態，可以處理複雜的運算。
  • 移植性高： 只要有對應的編譯器，C程式碼稍作修改（或无需修改）即可在不同平台上編譯執行。
  • 模組化設計： 支援函式的使用，可以將程式分割成多個模組，易於管理和維護。

2-2 C程式的結構

一個基本的C程式主要由以下幾個部分組成：

  1. 前置處理指令 (Preprocessor Directives)： 例如 #include，用於在編譯前引入標頭檔。
  2. 全域宣告 (Global Declarations)： 在所有函式之外宣告的變數或函式。
  3. main() 函式： 每個C程式都必須有一個 main() 函式，它是程式執行的起點。
  4. 使用者自訂函式 (User-defined Functions)： 由程式設計師自行定義的函式，用來執行特定任務。
  5. 註解 (Comments)： 用來說明程式碼，不會被編譯器執行。C語言支援兩種註解方式：// 單行註解 和 /* 多行註解 */。

2-3 第一個C程式：Hello World

學習任何程式語言的第一步通常是撰寫一個「Hello, World!」程式。以下是一個簡單的C程式範例：


#include <stdio.h>  // 引入標準輸入輸出標頭檔

/*
 這是一個多行註解的範例。
 main函式是程式的進入點。
*/
int main() {
    // printf() 函式用於將文字輸出到螢幕
    printf("Hello, World!\n");

    return 0;  // 表示程式正常結束
}

程式碼說明：

  • #include <stdio.h>：告訴編譯器引入標準輸入輸出函式庫的資訊。stdio.h 是一個標頭檔，其中包含了 printf() 等函式的宣告。
  • int main()：main 是函式名稱，代表主函式。int 表示此函式執行完畢後會回傳一個整數值。括號 () 表示這是一個函式。
  • { ... }：大括號標示出函式的範圍，也稱為函式主體。
  • printf("Hello, World!\n");：這是一個C敘述。printf() 是一個標準庫函式，用於將引號內的字串（以及可能的變數值）輸出到螢幕上。\n 是一個跳脫序列，代表換行。
  • return 0;：表示 main 函式執行成功並結束，回傳值0給作業系統。通常，回傳0代表程式正常結束，非0值代表出現錯誤。
  • C語言的每個敘述都必須以分號 ; 結尾。

2-4 編譯與執行C程式

C程式的執行過程通常包括以下步驟：

  1. 編輯 (Edit)： 使用文字編輯器撰寫C原始程式碼，並存成副檔名為 .c 的檔案（例如 hello.c）。
  2. 編譯 (Compile)： 使用C編譯器（例如 GCC, Clang）將原始程式碼轉換為機器碼（目的檔，通常副檔名為 .o 或 .obj）。編譯過程會檢查語法錯誤。 
    
gcc -c hello.c -o hello.o
  3. 連結 (Link)： 將編譯產生的目的檔與所需的函式庫連結起來，產生可執行檔（在Windows上副檔名為 .exe，在Linux/macOS上通常沒有副檔名）。 
    
gcc hello.o -o hello

    也可以將編譯和連結合併為一步：

    
gcc hello.c -o hello
  4. 執行 (Execute)： 執行產生的可執行檔。 
    
./hello  # 在 Linux/macOS 上
hello.exe # 在 Windows 命令提示字元上

重點提示：

  • 原始程式檔：副檔名為 .c。
  • 目的檔：編譯後產生，包含機器碼但尚未連結。
  • 執行檔：連結後產生，可以直接執行。
